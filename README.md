## 1. Project Overview
This project presents a complete design of a **Wireless Sensor Network (WSN)** for smart greenhouse monitoring in agriculture. The system enables continuous real-time monitoring of critical environmental parameters to optimize plant growth, reduce resource wastage, and enable automated control.

### Monitored Parameters
- Temperature  
- Humidity  
- Soil Moisture  
- Light Intensity  
- CO₂ Concentration  

---

## 2. Real-World Problem
Greenhouses require precise control of environmental conditions for healthy plant growth. Traditional manual monitoring is:
- Labor-intensive  
- Prone to human error  
- Inefficient in resource utilization  

This WSN-based solution addresses these challenges by providing automated, scalable, and energy-efficient monitoring.

---

## 3. Network Architecture
**Topology**: Hybrid (Cluster-based Mesh)

### Architecture Description
- Sensor nodes are grouped into clusters (5–10 nodes per cluster)
- Each cluster elects a **Cluster Head (CH)** for data aggregation
- Cluster Heads form a mesh network for multi-hop communication
- Gateway/Sink nodes connect the WSN to cloud platforms or local servers

### Why This Topology?
| Advantage                  | Benefit                              |
|---------------------------|--------------------------------------|
| Clustering                | Reduces energy consumption           |
| Data Aggregation          | Minimizes redundant transmissions    |
| Mesh Connectivity         | Provides fault tolerance             |
| Multi-hop Routing         | Improves coverage in large greenhouses |
| Scalability               | Supports expansion to larger farms   |

---

## 4. Key Benefits of the System
- Real-time environmental monitoring  
- Automated irrigation, ventilation, and lighting control  
- Early detection of pests, diseases, and equipment failures  
- Significant reduction in water and energy consumption  
- Scalable design suitable for commercial greenhouses
